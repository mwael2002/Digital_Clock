# Digital-Clock
A Digital Clock that is showed on LCD that Counts time in AM/PM format and the initial time is entered by the user on keypad.
Eclipse IDE is used to develop the project.

## Software Drivers:
#### 1- DIO
#### 2- Timer
#### 3- LCD
#### 4- Keybad

## Hardware:
### AVR Atnega32, LCD and Keybad.
